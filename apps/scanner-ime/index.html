Supported Barcode Formats

Our advanced scanner engine supports the following major formats:

  • Aztec
  • Codabar
  • Code 128
  • Code 39
  • Code 93
  • Data Matrix
  • EAN-8 & EAN-13
  • ITF
  • MaxiCode
  • PDF 417
  • QR Code
  • RSS-14 & RSS-Expanded
  • UPC-A & UPC-E

How It Works

  1. Enable "Barcode Keyboard" in your Android Language & Input settings.
  2. Select it as your active input method (IME).
  3. Open any app and tap a text field.
  4. Press the scan key on the keyboard to capture any barcode or QR code.

Why Choose Barcode Keyboard?

This app is designed for professionals and power users who need efficiency. By combining a traditional soft keyboard with a high-speed barcode scanner, we eliminate the need for expensive external hardware. It allows for rapid batch scanning and error-free data entry.
